# The Project

An Adventure/Survival/\<insert genre here\> game foundation. 

## What is it? 

An open source foundation for Unreal Engine games, with as much as possible implemented in C++ rather than blueprints.

## Features (Current and Future) 

- GAS
   - Most games need some system for attributes, abilities and effects. The Gameplay Ability System is pretty complex but handles all of it, including helping you with the networked parts. 

- Environment System
   - A custom built environment system for the passing of time/date, handling weather, sun/moon rotation and whatever else I might want to throw in there.

- Character Setup
   - Setup of characters, both player and AI driven ones (to be added), with Attribute Sets, Player States and Controllers. 

- Inventory
   - What's an adventure game without items and an inventory? A basic implementation exists for now, needs more work.
 
- Logging
   - A custom logger meant to be a little more useful than the default one. Will hopefully be connected to a logging system being built in Go! 

## But why..? 

It's primary purpose is for me to learn everything I can about Unreal Engine; its quirks, the good and bad, what I can and cannot do. 
If it also helps someone else, that's great!
